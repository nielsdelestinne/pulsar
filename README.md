# Pulsar

Apache Pulsar: Pub-Sub messaging system

## Usage

1. Run the script in `docker_pulsar.bat`, it will instruct Docker to set up the Pulsar Standalone mode.
2. Run the `Consumer`
3. Run the `Producer`.
